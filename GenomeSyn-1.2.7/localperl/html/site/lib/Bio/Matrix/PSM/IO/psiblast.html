NAME

Bio::Matrix::PSM::IO::psiblast - PSM psiblast parser

SYNOPSIS

See Bio::Matrix::PSM::IO for documentation

DESCRIPTION

Parser for ASCII matrices from PSI-BLAST (blastpgp program in BLAST distribution).
