NAME

Bio::Matrix::MatrixI - An interface for describing a Matrix

SYNOPSIS

  # Get a Matrix object

DESCRIPTION

This is an interface describing how one should be able to interact with a matrix. One can have a lot of information I suppose and this outline won't really work for PWM or PSSMs. We will have to derive a particular interface for those.
