NAME

Bio::Coordinate::Result::Match - Another name for Bio::Location::Simple

SYNOPSIS

  $loc = Bio::Coordinate::Result::Match->new(-start=>10,
                                            -end=>30,
                                            -strand=>+1);

DESCRIPTION

This is a location class for coordinate mapping results.
