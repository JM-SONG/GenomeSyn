NAME

Bio::Tools::Primer::AssessorI - interface for assessing primer pairs

SYNOPSIS

    use Bio::Tools::Primer::AssessorI;
    
    if( $obj->isa('Bio::Tools::Primer::AssessorI') ) {
        my $score = $obj->assess($primer_pair);
    }

DESCRIPTION

The Primer Assessor interface provides a interface for scoring functions of primer pairs to comply to. It is mainly used by Bio::Tools::Primer::Design module
