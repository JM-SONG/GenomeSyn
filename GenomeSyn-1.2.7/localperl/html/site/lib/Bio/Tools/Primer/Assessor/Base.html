NAME

Bio::Tools::Primer::Assessor::Base - base class for common assessor things

SYNOPSIS

    use Bio::Tools::Primer::Assessor::Base
    
    $base->weight(10);

DESCRIPTION

Base class for assessors, probably only defining the weight function
