NAME

Bio::Tools::Run::Maq::Config - Configuration data for maq commands

SYNOPSIS

Used internally by Bio::Tools::Run::Maq.

DESCRIPTION

This package exports information describing maq commands, parameters, switches, and input and output filetypes for individual maq commands.
